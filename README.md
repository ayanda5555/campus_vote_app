# Campus Vote App 🗳️

A Flutter-based campus governance and voting platform.

## Features
- ✅ Student authentication
- ✅ Real-time voting
- ✅ Policy management
- ✅ Admin dashboard

## How to Run
1. Clone the repo (git clone https://github.com/YOUR_USERNAME/campus_vote_app.git)
2. Run `flutter pub get`
3. Run `flutter run`
